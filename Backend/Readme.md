# 🤖 AI-Powered Smart Recruitment System

**Level:** 🔥 Hard  
**Tech Stack:** MERN + AI/ML Microservices  

---

## 📌 Overview

AI-Powered Smart Recruitment System ek full-stack hiring platform hai jo Artificial Intelligence ka use karke recruitment process ko automate karta hai.  
System resumes ko analyze karta hai, job description se match karta hai, candidates ko rank karta hai aur AI-based interview support provide karta hai.

---

## 🚀 Core Features

### 🔹 Job Management
- Company job postings
- Role-wise hiring workflow

### 🔹 Resume Processing
- PDF resume upload
- Resume parsing using NLP
- Skills, experience & education extraction

### 🔹 AI Resume Scoring
- Job Description vs Resume similarity
- Embedding-based matching
- Automated candidate shortlisting

### 🔹 Skill Gap Detection
- Missing skills identification
- Job-ready analysis per candidate

### 🔹 AI Interview Support
- Candidate-specific interview questions
- LLM-based question generation

### 🔹 Candidate Ranking Dashboard
- Resume score
- Skill match percentage
- Final AI ranking

### 🔹 Video Interview Analysis
- Video upload
- Speech-to-text conversion
- Keyword extraction
- Sentiment analysis

---

## 🧠 AI Integration

- **NLP:** Resume parsing & skill extraction  
- **Embeddings:** Resume–JD similarity matching  
- **LLM:** Interview question generation  
- **Speech-to-Text:** Video interview transcription  
- **Sentiment Analysis:** Confidence & tone detection
